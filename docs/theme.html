CSS Variables

The theme is defined by the following CSS variables:

--font-family: Verdana, sans-serif;
--font-monospace: "Courier New", monospace;
--font-size: 14px;
--font-weight: 400;

--color-text: black;
--color-background: white;

--color-primary: blue;
--color-secondary: gray;
--color-danger: red;
--color-success: green;

--filter-focus: opacity(.6);
--filter-active: brightness(1.1);
